# User Story: Discount Code Application

As a user,

I want to reset my password

So that I can regain access to my account.

## Acceptance Criteria

1. Email is mandatory.

2. Email must exist.

3. OTP is valid for 5 minutes.

4. OTP can be used only once.

5. Password minimum length is 8.

6. Password must contain uppercase, lowercase, digit, and special character.

7. New password cannot match the previous password.

8. Maximum 3 OTP attempts.

9. Expired OTP should display an error.
